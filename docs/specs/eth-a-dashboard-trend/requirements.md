## 目标
- 将“A 仪表盘 v2.0.1 全部内容”落地为可执行的日度看板。
- 点击“今日运行”生成当日输出，并沉淀历史趋势。
- 运行顺序严格按 G0→V1→V2→...→最终动作映射执行，前者具备否决权。

## 核心输出（面板必须可见）
- 三档状态：A / B / C
- 当前 β、β_cap
- 是否对冲：开/关（C 档默认轻对冲）
- SC-Phase 相位标签：Up-Mid / Late-Div / BTD→BTR
- 相位置信度（受 SVC 与分发闸门加成修正）
- 是否允许极限重仓（需 BE+三域+ΔES+结构强分数）
- 触发原因 Top3

## 运行顺序与硬闸门
- G0 宏观总闸门（最高优先级）
- V1 流动性总分（FoF）
- V2 风险开关（Risk ON/OFF）
- V3 杠杆结构与清算博弈（三件套）
- V4 卖墙与浮筹结构
- V5 ETF 买墙与连击风控 + 突破验证
- V6 风险矩阵与危险信号系统
- V6.2 SVC 结构性价值捕获模块
- V7 买家阶段指标（BPI）
- V8 牛熊条件矩阵（BCM）
- HPM 历史相位映射器
- SC-Phase 相位判定
- BE 相变判定
- 三域扫描（ΔES=0.6）
- 最终动作映射（A/B/C + β + 对冲 + β_cap）

## 数据与历史趋势
- 日度运行产生一条完整记录（输入快照 + 输出 + 触发原因 + 置信度）。
- 历史趋势至少展示：状态、β、置信度、FoF。

## 体验要求
- 看板式布局，清晰展示状态与闸门链路结果。
- UI 需要视觉上“有设计感”，非模板化。
- 本地可运行（无需后端），仅支持手动输入/导入数据。
- 闸门链路必须可展开显示完整计算与命中规则，支持审计。
- 支持公开数据源自动抓取（宏观/稳定币/ETF 流入），并标注数据来源。
